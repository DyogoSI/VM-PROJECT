O que é uma maquina virtual?

Uma maquina virtual (VM) é um software que simula um computador fisico.Ela permite executar sistemas operacionais e aplicativos de forma isolada,como se fosse uma maquian real.

Vantagens

  • Executar varios sistemas operacionais no mesmo computador.
  • Ideal para testes sem afetar o sistema principal
  • Backup e restauração faceis

Desvantagens

  • Consome bastante memoria e processamento.
  • Desempenho inferior ao sistema instalado direto no PC.

Exemplos de softwares

  • VMware Workstation
  • VirtualBox
  • Hyper-V
  • Parallels Desktop

Beneficios das maquinas Virtuais

Segurança

Ambientes isolados garantem testes seguros sem afetar seu sistema real

Flexibilidade

Você pode restaurar snapshots e alternar sistemas com facilidade

Infraestrutura

Ideal para simular servidores e redes completas e um só PC.

Perguntas Frequentes

É um ambiente simulado dentro de um computador fisico, que permite rodar sistemas operacionais de forma isolada

Não necessariamente.Depende do sistema que será virtulizado,mas quanto mais memória e CPU, melhor.

Todas as maquinas vituais tem uma quantidade de giga especifica,é recomendado colocar a quantidade que o software aonde você estiver instalando o sistema operacional permitir,mas não coloque um tamanho muito grande pra não ocupar espaço no sistema operacional principal da sua maquina
